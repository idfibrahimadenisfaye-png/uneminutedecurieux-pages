Privacy Policy — UneMinuteDeCurieux (Publication automatique)

Last updated: July 2026

This is a personal, single-user automation tool built by an individual creator to generate and publish short-form video content to their own TikTok account (UneMinuteDeCurieux).

Data collected

This tool does not collect, store, or share any personal data from TikTok users other than the account owner. It only accesses the TikTok account it is explicitly authorized to post to, via TikTok's official OAuth authorization flow.

Data usage

Any data accessed via the TikTok API (such as account info needed to publish content) is used solely to publish video content to the authorized account. No data is sold, shared with third parties, or used for advertising.

Data retention

No TikTok user data is stored beyond what is temporarily needed to complete a publishing request.

Contact

For any question regarding this privacy policy, please contact the account owner directly via TikTok (@UneMinuteDeCurieux).
